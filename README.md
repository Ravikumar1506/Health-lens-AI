# HealthLens AI 🏥
### AI-Powered Medical Health Report Analyzer

## 🚀 How to Run

**Locally:**
1. Download this repo as ZIP → Extract it
2. Open the extracted folder
3. Double-click `index.html` in Chrome — done!

> All 3 files (index.html, style.css, script.js) must stay in the same folder.

## 💡 Features

- 🔬 Real OCR text extraction (Tesseract.js)
- 📊 60+ health parameters with reference ranges
- 💯 Health Score (0–100) and Risk Level
- 📈 4 interactive charts (Chart.js)
- 🤖 AI Chat
- 🥗 Diet, Exercise & Lifestyle recommendations
- 📄 PDF export of full analysis
- 🌙 Dark / Light mode
- 💾 Report history saved in browser
- 📱 Fully responsive design
